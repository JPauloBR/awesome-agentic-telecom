# Awesome Agentic Telecom

A curated collection of specifications, tools, research, and practical references at the intersection of agentic AI and telecommunications.

Contributions are welcome. Resources should be technically substantive, publicly accessible, and directly useful to people building or operating telecom AI systems.

## Protocols and interoperability

- [Model Context Protocol](https://modelcontextprotocol.io/) — open protocol for connecting AI applications to tools and data
- [MCP specification](https://modelcontextprotocol.io/specification/latest) — normative protocol documentation
- [Agent2Agent Protocol](https://github.com/a2aproject/A2A) — open protocol for communication between independent agent systems

## Agent frameworks

- [LangGraph](https://github.com/langchain-ai/langgraph) — stateful orchestration for long-running agents
- [Microsoft AutoGen](https://github.com/microsoft/autogen) — framework for agentic applications
- [Semantic Kernel](https://github.com/microsoft/semantic-kernel) — SDK for integrating models, tools, and processes

## Evaluation and observability

- [Arize Phoenix](https://github.com/Arize-ai/phoenix) — open-source tracing and evaluation for AI applications
- [LangSmith](https://docs.smith.langchain.com/) — tracing, evaluation, and deployment tooling
- [OpenTelemetry](https://opentelemetry.io/) — vendor-neutral telemetry framework

## Telecom standards and industry resources

- [TM Forum Open APIs](https://www.tmforum.org/oda/open-apis/) — standardized telecom business and operational APIs
- [TM Forum Autonomous Networks](https://www.tmforum.org/autonomous-networks/) — industry framework for network automation
- [ETSI Experiential Networked Intelligence](https://www.etsi.org/technologies/experiential-networked-intelligence) — standards work for AI-driven network management
- [3GPP specifications](https://www.3gpp.org/specifications-technologies) — mobile telecommunications specifications
- [Broadband Forum](https://www.broadband-forum.org/) — standards and resources for broadband networks

## Suggested contribution areas

- Customer-care and troubleshooting agents
- Network operations and assurance
- Broadband gateway and home-network diagnostics
- Telecom knowledge graphs and retrieval
- Multi-agent interoperability
- Safety, governance, evaluation, and human escalation

## Contributing

Open an issue or pull request with the resource, a one-sentence explanation, and the section where it belongs. Promotional links without clear technical value will not be accepted.
